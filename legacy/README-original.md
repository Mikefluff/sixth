# SIXTH - Минималистичный язык программирования

**SIXTH** - это демонстрация самосборки программирования от 15 базовых примитивов до полноценной среды разработки ИИ. Проект показывает, как из минимального набора операций можно построить сложные системы через пошаговую самосборку.

## 🎯 Концепция

Основная идея - доказать, что **ВСЕГО 15 примитивов** достаточно для создания любой программы, включая системы искусственного интеллекта. Каждый уровень строится исключительно из предыдущих уровней.

## 🏗️ Архитектура

### Уровень 0: 15 Базовых примитивов

**Стековые операции:**
- `dup` - дублировать верхний элемент
- `drop` - удалить верхний элемент  
- `swap` - поменять два верхних элемента
- `over` - копировать второй элемент наверх

**Арифметика:**
- `+` `-` `*` `/` `mod` - основные операции
- `=` `<` `>` - сравнения

**Память:**
- `store` - сохранить в переменную
- `load` - загрузить из переменной

**Управление:**
- `eval` - выполнить код
- `:` `;` - определение функций
- `if` `else` `then` - условия

### Уровень 1: core.6th (40 функций)

Построены **ТОЛЬКО** из 15 примитивов:

**Константы:**
```forth
: zero 0 ;
: one 1 ;
: two 2 ;
...
: ten 10 ;
```

**Стековые операции:**
```forth
: 2dup over over ;
: 2drop drop drop ;
: nip swap drop ;
: tuck swap over ;
```

**Логика:**
```forth
: not 0 = ;
: true 1 ;
: false 0 ;
```

**Арифметика:**
```forth
: 1+ 1 + ;
: 1- 1 - ;
: 2* 2 * ;
: abs dup 0 < if 0 swap - then ;
: factorial dup 1 > if dup 1- factorial * else drop 1 then ;
```

## 📖 Руководство программиста

### Основные принципы

1. **Только 15 примитивов** - никаких расширений
2. **Самосборка** - каждый уровень из предыдущего
3. **Стековое программирование** - данные на стеке
4. **Постфиксная нотация** - операторы после операндов

### Синтаксис

**Определение функций:**
```forth
: имя-функции код... ;
```

**Условия:**
```forth
условие if код-если else код-иначе then
```

**Переменные:**
```forth
значение "имя" store    \ сохранить
"имя" load              \ загрузить
```

**Примеры:**
```forth
\ Квадрат числа
: square dup * ;

\ Факториал
: factorial 
  dup 1 > if 
    dup 1- factorial * 
  else 
    drop 1 
  then ;

\ Условная функция
: abs dup 0 < if 0 swap - then ;
```

### Работа со стеком

```forth
42          \ [42]
dup         \ [42, 42] 
3           \ [42, 42, 3]
+           \ [42, 45]
swap        \ [45, 42]
drop        \ [45]
```

## 🎯 Философия проекта

### Самосборка (Self-Bootstrapping)

Проект демонстрирует **радикальный минимализм**:
1. **15 примитивов** → все остальное
2. **Каждый уровень** строится только из предыдущих
3. **Никаких встроенных функций** кроме примитивов
4. **Полная прозрачность** - весь код видим

### Путь к ИИ

**15 примитивов** → **40 функций** → **структуры данных** → **объекты** → **экспертные системы** → **нейросети** → **ИИ**

Каждый шаг - **самосборка** без внешних зависимостей.

### Практическое применение

- **Обучение** - понимание основ программирования
- **Исследования** - минимальные вычислительные модели  
- **Прототипирование** - быстрая проверка идей
- **Философия** - что такое "программирование" в чистом виде
